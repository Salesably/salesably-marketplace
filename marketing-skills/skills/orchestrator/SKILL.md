# Orchestrator

This skill acts as a routing system for marketing projects—diagnosing needs, recommending the right skills, and sequencing them for comprehensive campaign execution.

## Objective

Help users navigate the marketing skills suite by identifying the right skill(s) for their needs and sequencing them effectively for multi-step projects.

## The 10 Marketing Skills Available

### Foundation Layer
| Skill | Purpose | Use When |
|-------|---------|----------|
| `brand-voice` | Define consistent communication | Starting from scratch, content sounds generic, need consistency |
| `positioning-angles` | Find market differentiation | New product, competitive market, unclear positioning |

### Strategy Layer
| Skill | Purpose | Use When |
|-------|---------|----------|
| `keyword-research` | Identify content opportunities | Building content strategy, filling calendar, finding topics |
| `lead-magnet` | Create opt-in offers | Growing email list, launching product, need leads |

### Execution Layer
| Skill | Purpose | Use When |
|-------|---------|----------|
| `direct-response-copy` | Write conversion-focused copy | Sales pages, landing pages, any conversion goal |
| `seo-content` | Create search-optimized articles | Blog posts, pillar content, organic traffic goals |
| `newsletter` | Build recurring engagement | Regular audience communication, email content |
| `email-sequences` | Automate customer journeys | Welcome flows, launches, conversions, onboarding |

### Distribution Layer
| Skill | Purpose | Use When |
|-------|---------|----------|
| `content-atomizer` | Repurpose across platforms | Maximizing content ROI, filling calendars, multi-platform |

## Diagnostic Questions

To identify the right skill, ask:

### 1. What's the primary goal?
- **Build awareness** → `seo-content`, `content-atomizer`, `newsletter`
- **Generate leads** → `lead-magnet`, `direct-response-copy`, `email-sequences`
- **Convert to sales** → `direct-response-copy`, `email-sequences`
- **Retain customers** → `email-sequences`, `newsletter`
- **Define brand** → `brand-voice`, `positioning-angles`
- **Plan content** → `keyword-research`

### 2. What assets exist already?
- **Nothing (starting fresh)** → Start with `brand-voice`, then `positioning-angles`
- **Brand guidelines only** → Move to strategy layer
- **Some content exists** → `content-atomizer` to maximize it
- **Email list exists** → `email-sequences` or `newsletter`
- **Traffic exists** → `lead-magnet` to capture it

### 3. What's the timeline?
- **Immediate results** → `direct-response-copy`, `lead-magnet`
- **This quarter** → `seo-content`, `email-sequences`, `newsletter`
- **Long-term foundation** → `brand-voice`, `positioning-angles`, `keyword-research`

### 4. What's the primary channel?
- **Organic search** → `seo-content`, `keyword-research`
- **Email marketing** → `email-sequences`, `newsletter`, `lead-magnet`
- **Social media** → `content-atomizer`, `newsletter` (repurposing)
- **Landing pages/ads** → `direct-response-copy`, `lead-magnet`
- **Multiple channels** → Start with `brand-voice` for consistency

## Sequencing Playbooks

Common multi-skill workflows:

### Playbook 1: New Product Launch
**Goal**: Take a new product to market
**Timeline**: 4-8 weeks

```
Week 1-2:    brand-voice → Define how we sound
                 ↓
Week 2-3:    positioning-angles → Find differentiation
                 ↓
Week 3-4:    lead-magnet → Build pre-launch list
                 ↓
Week 4-5:    direct-response-copy → Create sales page
                 ↓
Week 5-6:    email-sequences (launch) → Build launch sequence
                 ↓
Week 6+:     content-atomizer → Promote across channels
```

### Playbook 2: Content Engine
**Goal**: Build sustainable organic traffic
**Timeline**: Ongoing, quarterly planning

```
Quarter Start:  keyword-research → Identify Q's topics
                     ↓
Ongoing:        seo-content → Create pillar articles
                     ↓
Weekly:         content-atomizer → Repurpose for social
                     ↓
Weekly:         newsletter → Share with subscribers
                     ↓
As needed:      lead-magnet → Capture traffic as leads
```

### Playbook 3: Lead Generation System
**Goal**: Build email list and nurture to conversion
**Timeline**: 2-4 weeks setup, ongoing operation

```
Week 1:      brand-voice → Ensure consistent messaging
                 ↓
Week 1-2:    lead-magnet → Create opt-in offer
                 ↓
Week 2:      direct-response-copy → Build landing page
                 ↓
Week 2-3:    email-sequences (welcome) → Build welcome flow
                 ↓
Week 3-4:    email-sequences (nurture) → Build nurture flow
                 ↓
Week 4:      email-sequences (conversion) → Build sales sequence
```

### Playbook 4: Brand Refresh
**Goal**: Establish clear brand identity and voice
**Timeline**: 2-3 weeks

```
Week 1:      brand-voice → Define new voice guidelines
                 ↓
Week 1-2:    positioning-angles → Clarify differentiation
                 ↓
Week 2-3:    Apply voice to existing:
             - direct-response-copy (update pages)
             - email-sequences (update flows)
             - newsletter (align templates)
             - seo-content (audit existing)
```

### Playbook 5: Email Marketing Setup
**Goal**: Build complete email marketing system
**Timeline**: 3-4 weeks

```
Week 1:      lead-magnet → Create opt-in offer
                 ↓
Week 1-2:    direct-response-copy → Landing page for magnet
                 ↓
Week 2:      email-sequences (welcome) → Welcome flow
                 ↓
Week 2-3:    email-sequences (nurture) → Nurture flow
                 ↓
Week 3-4:    newsletter → Set up regular broadcasts
                 ↓
As needed:   email-sequences (conversion) → Sales sequences
```

## Handoff Guidance

When moving between skills, pass this context:

### From → To Context Transfer

**brand-voice → All other skills**
- Voice profile (4 dimensions)
- This-but-not-that guidelines
- Vocabulary do's and don'ts

**positioning-angles → direct-response-copy**
- Chosen positioning statement
- Key differentiators
- Competitive context

**keyword-research → seo-content**
- Target keyword
- Search intent analysis
- Competitor content gaps

**lead-magnet → email-sequences**
- Lead magnet topic/promise
- Audience expectations
- Natural next steps

**seo-content → content-atomizer**
- Full article content
- Key insights to extract
- Target platforms

## Output Format

When diagnosing needs, provide:

1. **Situation Assessment**: Summary of the user's current state and goals
2. **Recommended Skill(s)**: Primary and supporting skills
3. **Sequencing Plan**: Order of operations if multiple skills needed
4. **Timeline Estimate**: Rough phases for the work
5. **Quick Start**: First action to take

## How to Use This Skill

This skill (`orchestrator`) is the starting point when:
- You're unsure which skill to use
- You have a complex, multi-faceted marketing goal
- You want to build a comprehensive marketing system
- You're planning a major campaign or launch

After diagnosis, invoke the recommended skill(s) directly for detailed execution.
